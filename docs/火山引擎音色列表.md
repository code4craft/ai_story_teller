# 火山引擎在线音色列表

注：

1. 下述音色列表中，中文音色可支持中英文混合场景。
2. 病弱少女、 活泼女孩、和蔼奶奶、邻居阿姨四个音色暂不支持双向流式接口调用。
3. moon系列音色，会有概率出现音色跳变/突变问题；
4. 少量mars系列音色，会有稳定性问题，如「解说男生（jieshuonansheng）」；
5. moon系列音色，暂不支持语种参数指定功能。
